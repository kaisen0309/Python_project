# **小狗的逆襲–結合臉部辨識與聲音辨識的小遊戲**

此專題為四人共同製作，利用Python語言，結合pygame、speech_recognition
和mediapipe等套件，設計一款結合臉部辨識與聲音辨識的小遊戲。 <br>
玩家在遊玩時需要透過擺頭的方式，操作遊戲角色的移動，並透過喊出指定的字詞，清除路上的障礙物。
